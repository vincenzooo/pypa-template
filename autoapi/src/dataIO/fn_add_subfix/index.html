src.dataIO.fn_add_subfix.fn_add_subfix(fileName, subfix='', newext=None, strip=False, pre='')

Add a subfix to a filename. A new extension can be defined (dot must be included). if newext is None, leaves extension unchanged (use empty string to remove extension). If strip is set, only file name (not including path) is returned. If pre is passed, it is added as prefix (before the file name, excluding the leading directory). datafile=’file_with_data.txt’ outfolder=’output_directory’ print fn_add_subfix(datafile,’_output’) print fn_add_subfix(datafile,’_output’,’.dat’) import os fn_add_subfix(os.path.join(outfolder,datafile),’_output’,’.dat’)
