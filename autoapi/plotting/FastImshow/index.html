plotting.FastImshow

Fast Plotter for Large Images - Resamples Images to a target resolution on each zoom. Example:

sz = (10000,20000) # rows, cols
buf = np.arange(sz[0]*sz[1]).reshape(sz)
extent = (100,150,1000,2000)
fig = plt.figure()
ax  = fig.add_subplot(111)
im = FastImshow(buf,extent,ax)
im.show()
plt.show()
